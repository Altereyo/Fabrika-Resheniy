# Фабрика Решений

## Для начала вам нужно установить Node.js
```
https://nodejs.org/en/
```

## Далее откройте папку с проектом через терминал и установите пакеты NPM
```
npm install
```

## Далее запустите сервер и дождитесь надписи ***Server running at:***
```
npm run express:run
```
## Теперь можете запускать проект
```
npm run serve
```
